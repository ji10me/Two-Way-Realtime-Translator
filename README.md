# Two-Way Realtime Translator (Moonshine + Whisper Hybrid)

VRChatやその他コミュニケーションツールでの利用を想定した、ローカルで動作する双方向リアルタイム音声翻訳アプリです。
英語・スペイン語の音声認識に超高速な **Moonshine (ONNX版)**、日本語等の音声認識に高精度な **Whisper** を使用するハイブリッド構成を採用しています。

---

## 主な特徴・仕様

1. **爆速・ループなしのリスニング (相手の音声認識)**
   * 相手の音声が **「英語 (English)」** または **「スペイン語 (Spanish)」** の場合、超軽量で遅延が極めて少なく、無音時の幻覚（同じ言葉を繰り返すループ暴走）が発生しない **Moonshine (ONNX版)** を使用して音声認識を行います。
   * 英語・スペイン語以外の言語が選択された場合は、自動的に多言語対応の **Faster-Whisper** へ切り替わります（フォールバック機能）。
2. **高精度なスピーキング (自分の音声認識)**
   * 自分の音声（日本語推奨）は、高い認識精度を誇る **Faster-Whisper** を使用してテキスト化し、翻訳して出力します。
3. **VRChat ChatBox (OSC送信) のコントロール**
   * スピーキング設定の下部にある **「VRChat ChatBox (OSC) に送信する」** のチェックボックスで、VRChatのチャットボックスへテキストを直接送信するかどうかを制御できます。ON/OFF状態は自動で保存されます。
4. **デバイスの自動名指し追従**
   * マイクやスピーカーのデバイス設定はインデックス番号（`[0]` や `[3]` など）ではなく、デバイスの「名前」で記憶します。デバイスの抜き差しでシステム上の番号が変わっても、自動的に正しいデバイスを再検索して接続します。
5. **PC依存のない自動セットアップ (`launcher.py` / `run.bat`)**
   * 仮想環境（`venv`）の自動構築や、不足している CUDA / cuDNN の DLL 自動配備機能を搭載しています。

---

## 使い方・起動手順

1. **通常起動**
   * フォルダ内の **`run.bat`** (推奨) または `VRC_Translator.exe` を実行します。
   * 初回起動時や環境変更時は、裏で必要なライブラリの自動セットアップが走ります（C++ビルド環境がないPCでも、自動的にCPU版のLLMにフォールバックして正常起動します）。
2. **デバイスと翻訳設定**
   * 「相手の音声」および「自分の音声」で、それぞれ使用するオーディオデバイスを選択します。
   * 相手が話す言語（認識言語）と、自分が話す言語（認識言語）、それぞれの翻訳先（ターゲット言語）を選択します。
3. **翻訳の開始**
   * **「翻訳開始 (Start)」** ボタンをクリックします。モデルのロード完了後、翻訳処理が開始されます。
   * 終了する時は **「停止 (Stop)」** をクリックするか、ウィンドウを閉じてください。

## 他のPCへの移行方法

他のPCにアプリをコピーして実行する場合、以下のファイルおよびフォルダのみを移行先にコピーしてください。

### 〇 コピーが必要なファイル
* `VRC_Translator.exe` (または `run.bat`, `run_log.bat`)
* `main.py`
* `requirements.txt`
* `config.json` (任意)
* `models/` フォルダ (翻訳モデル `Hy-MT2-1.8B` やスペイン語 Moonshine モデルが内包されています)

### × コピーしてはいけないフォルダ
* **`venv/` フォルダ** (コピー元のPC環境に依存しているため、コピーするとエラーの原因になります。移行先で初回起動した際に自動生成されます)
* `build/`, `dist/` フォルダ (EXEビルド時の一時ファイルのため不要です)

## システム構成とモデル情報
* **音声翻訳 (MT)**: `Tencent/Hy-MT2-1.8B-GGUF` (CPU/GPU両対応・日本語/英語/スペイン語/中国語/韓国語/ロシア語双方向)
* **相手音声認識 (STT)**: 
  * `useful-sensors/moonshine/tiny` (英語)
  * `UsefulSensors/moonshine-es` (スペイン語)
  * `faster-whisper (tiny)` (その他)
* **自分音声認識 (STT)**: `faster-whisper (tiny)`
